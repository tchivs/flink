traces.checkpoint.span-detail-level
 SPANS_PER_TASK

Enum

 Detail level for reporting checkpoint spans. Possible values: - BREAKDOWN_PER_CHECKPOINT (default): Sum/Max for submetrics per checkpoint - BREAKDOWN_PER_CHECKPOINT_WITH_TASKS: Sum/Max for submetrics per checkpoint and arrays of task aggregates. - BREAKDOWN_PER_TASK: Sub/Max for submetrics of checkpoint and tasks (tasks as child spans) - BREAKDOWN_PER_SUBTASK: Sub/Max for submetrics of checkpoint, tasks, and subtasks (tasks as child spans, subtasks as grand-child spans)

Possible values:
  • "SPANS_PER_CHECKPOINT"
  • "SPANS_PER_CHECKPOINT_WITH_TASKS"
  • "SPANS_PER_TASK"
  • "SPANS_PER_SUBTASK"
